# Website Prompt: Latinos Barbershop

**Goal:** Create a modern, stylish, and responsive landing page for "Latinos Barbershop". This is a demo site, so keep the implementation straightforward and simple (no backend, no DB, no testing).

**Tech Stack:**
- React (Vite)
- Tailwind CSS (for styling)
- AOS (Animate On Scroll) library for scroll animations
- Lucide React (for icons)
- `react-booking-calendly` (for appointments)
