# Website Prompt: La Fond Veterinary Hospital

**Goal:** Create a warm, welcoming, and professional landing page for "La Fond Veterinary Hospital". This is a demo site, so keep the implementation straightforward (no complex backend).

**Tech Stack:**
- React (Vite)
- Tailwind CSS (for styling)
- AOS (Animate On Scroll) library for scroll animations
- Lucide React (for icons)
- **`react-booking-calendly`** (REQUIRED for appointments)

**Assets Available (assume these exist in `public/assets/`):**
- `favicon.jpg` (IMPORTANT: Apply CSS `rounded-full` to make this a circular logo).
- `cute-small-dogs-banner.jpg`
- `funny-dog.jpg`
- `funny-dog-2.jpg`
- `funny-cat.jpg`
