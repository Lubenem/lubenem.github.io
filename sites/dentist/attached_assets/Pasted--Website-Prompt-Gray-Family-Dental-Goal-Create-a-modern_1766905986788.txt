# Website Prompt: Gray Family Dental

**Goal:** Create a modern, simple, and responsive landing page for "Gray Family Dental". This is a demo site, so keep the implementation straightforward (no complex backend).

**Tech Stack:**
- React (Vite)
- Tailwind CSS (for styling)
- AOS (Animate On Scroll) library for scroll animations
- Lucide React (for icons)

## Design Guidelines
- **Visual Style:** Clean, medical/professional but warm (blues, whites, soft grays).
- **Animations:** Use the AOS library to animate elements as they scroll into view (fade-up, zoom-in, slide-left/right).
- **Responsiveness:** Must look good on mobile and desktop.
- **Simplicity:** Do not overengineer. No backend database required. No automated tests.

## Implementation Steps
1. Initialize React + Vite project.
2. Install Tailwind CSS and AOS (`npm install aos`).
3. Create components for each section.
4. Add the provided dummy images (placeholders if files missing).
5. Ensure the Google Maps iframe works.
6. Verify scroll animations work smoothly.
