# Website Prompt: Military City Air Conditioning and Heating

**Goal:** Create a professional, reliable, and high-converting landing page for "Military City Air Conditioning and Heating". This is a demo site, so keep the implementation straightforward (no complex backend).

**Tech Stack:**
- React (Vite)
- Tailwind CSS (for styling)
- AOS (Animate On Scroll) library for scroll animations
- Lucide React (for icons)
- **`react-booking-calendly`** (REQUIRED for appointments)

**Assets Available (assume these exist in `public/assets/`):**
- `favicon.jpg` (IMPORTANT: Apply CSS `rounded-full` to make this a circular logo).
