## Packages
aos | For scroll animations (fade-up, fade-left)
react-day-picker | For the booking calendar component
date-fns | For date formatting and manipulation in the booking form
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes safely

## Notes
Integration with Google Maps requires the CID provided in the prompt.
The booking system is a custom implementation connecting to POST /api/appointments.
Bilingual support (EN/UA) is handled via React Context.
Images are imported from @assets/ as requested.
